# HLS Server

The server will be created according to [this documentation](https://developer.apple.com/documentation/http-live-streaming/deploying-a-basic-http-live-streaming-hls-stream)

## Usage

Run the following to start the server:

```sh
cargo run
```

Then, open [localhost](http://localhost:3000/) in your browser of choice to see
a simple `Hello world` message.

> [!NOTE]
> Once the server is finished, I'll write a Dockerfile to run it with docker.
